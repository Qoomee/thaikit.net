# ThaiKit

ThaiKit 是一个为在泰国生活、工作和创业的人们提供实用资源导航的网站。

## 功能特点

- 清晰的分类导航
- 响应式设计
- 现代化的用户界面
- 多语言支持

## 主要模块

1. Essentials（基础必备）
   - 签证信息
   - 电话卡/宽带
   - 租房平台
   - 本地银行
   - 医院资源
   - 保险资源

2. Lifestyle（生活方式）
   - 日常生活
   - 外卖平台
   - 交通出行
   - 演出展览
   - 电影院
   - 媒体新闻
   - 旅游观光

3. Work & Jobs（工作）
4. Business（商务）
5. Learning（学习）
6. About（关于）

## 技术栈

- HTML5
- CSS3
- JavaScript
- Font Awesome 图标

## 本地开发

1. 克隆仓库
```bash
git clone https://github.com/yourusername/thaikit.git
```

2. 打开项目
```bash
cd thaikit
```

3. 使用浏览器打开
```bash
open index.html
```
